# Optimise Mod for Minecraft 1.21.1 (Fabric)

一个为Minecraft 1.21.1设计的性能优化模组，提供多种优化选项和现代化的GUI界面。

## 功能特性

### 性能优化
- **实体渲染优化**: 减少实体渲染开销，包括距离剔除和LOD(细节层次)优化
- **粒子效果优化**: 限制粒子数量和渲染距离，减少粒子系统开销
- **方块渲染优化**: 优化方块渲染和地形更新频率
- **天气渲染优化**: 优化天空和天气效果渲染

### 高级选项
- **实体距离剔除**: 不渲染远距离的实体
- **实体LOD**: 对远距离实体使用简化渲染
- **粒子距离剔除**: 不渲染远距离的粒子
- **粒子数量限制**: 限制同时显示的粒子数量

### 内存优化
- **资源卸载**: 更积极地卸载未使用的资源
- **内存清理**: 定期清理内存中的不必要数据

### GUI优化
- **FPS显示**: 在游戏界面右上角显示当前FPS
- **现代化界面**: 采用选项卡式布局的配置界面
- **实时配置**: 无需重启游戏即可应用优化设置

## 使用方法

1. 安装模组到Minecraft 1.21.1
2. 启动游戏后，按`K`键打开优化配置界面
3. 在不同选项卡中调整优化设置
4. 点击"Apply"应用更改，或"Reset to Defaults"重置为默认值

## 配置选项

### 渲染选项卡
- **实体渲染优化**: 启用/禁用实体渲染优化
- **粒子优化**: 启用/禁用粒子效果优化
- **方块渲染优化**: 启用/禁用方块渲染优化
- **天气渲染优化**: 启用/禁用天气渲染优化

### 高级选项卡
- **实体距离剔除**: 启用/禁用实体距离剔除
- **实体LOD**: 启用/禁用实体细节层次优化
- **粒子距离剔除**: 启用/禁用粒子距离剔除
- **粒子数量限制**: 启用/禁用粒子数量限制

### 内存选项卡
- **资源卸载**: 启用/禁用资源卸载优化
- **内存清理**: 启用/禁用内存清理功能

### GUI选项卡
- **FPS显示**: 在游戏界面显示FPS

## 安装

1. 下载并安装 [Fabric Loader](https://fabricmc.net/use/) 适用于Minecraft 1.21.1
2. 下载最新版本的Optimise模组
3. 将模组文件放入`.minecraft/mods`文件夹
4. 使用Fabric配置文件启动Minecraft

## 从源代码构建

1. 克隆仓库
2. 运行 `./gradlew build` (Linux/Mac) 或 `gradlew.bat build` (Windows)
3. 构建的模组将在 `build/libs/` 目录中

## 兼容性

- Minecraft版本: 1.21.1
- Fabric Loader: 最新版本
- 依赖: Fabric API

## 注意事项

- 某些优化可能会影响游戏视觉效果
- 如果遇到问题，可以尝试禁用特定优化选项
- 建议根据电脑配置调整优化选项以获得最佳性能
